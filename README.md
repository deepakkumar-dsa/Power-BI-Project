# Power-BI-Project
🌾 Agriculture Data Analysis Dashboard

🔍 Project Overview

This project analyzes agricultural data to evaluate total yield, crop performance, and regional productivity. The Power BI dashboard provides interactive visualizations to identify trends, improve decision-making, and optimize agricultural output.

🛠️ Tools & Technologies

Power BI Desktop

Power Query Editor

DAX (Data Analysis Expressions)

Excel / CSV datasets

📊 Key Metrics

Farms count: 500 farms

Total Yield Kg: 2019kg

Average Yield: 4.022 kg/hectare

Average days to Harvest: 119 days

📊 Dashboard Highlights

Total yield distribution across regions

Crop-wise production comparison

Year-wise trend analysis

Region-wise performance tracking

Interactive filters for dynamic insights

📈 Key Insights

The highest total yield was recorded in Central USA with 437.43 kg

The top-performing crop is Soya Bean, contributing 23% of total production

South USA shows consistently low productivity, indicating scope for improvement


🎯 Project Outcome

Identified high and low-performing regions

Improved understanding of crop yield patterns

Enabled data-driven agricultural planning

Highlighted opportunities for increasing productivity
